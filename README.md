# Excel-Dashboard
It includes the sales and retail data 
It is based on the product sales and orders
It contains some calculations and formulas

1) Serial no
2) Date
3) Region
4) Saler Name
5) Quantity ordered
6) Quantity sales
7) Customer type
8) Customer Name
9) Unit Cost
10) Total Amount
11) CGST (12%)
12) SGST (8%)
13) Total Amount After GST
14) Selling price
15) Total Sales
16) Total Profit
